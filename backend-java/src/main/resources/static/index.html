游戏规则

点击你的领地选择军队，然后点击相邻格子移动
移动到自己的领地：源格子保留1兵力，目标格子获得总兵力-1
移动到空地：占领该格子，源格子保留1兵力
移动到敌人领地：进行战斗，兵力多者获胜
每2秒你的军队数量会增加
占领敌方将军（皇冠图标）来获胜

💡 优先保护你的将军
💡 扩张领土以获得更多军队
💡 只有相邻的敌人领地才可见
